# Root Access

- **Genre:** 2D action-platformer / metroidvania  
- **Core Fantasy:** You are a two-person team, cutting through obstacles using brains, agility, and improvisation to stop the impending apocalypse.  
- **Gameplay Hook:** Fast, fluid **character swapping** where each protagonist has distinct abilities and ability progression.

## Aesthetic & Tone

- **Art Style**
  - Pixel art with a blend of:
    - **Sci-fi:** terminals, panels, cables, antennae.
    - **Solar punk:** hopeful, green tech and sustainable design.
    - **Naturalistic forest elements:** roots, overgrowth, organic shapes reclaiming infrastructure.

- **Tone**
  - Generally **light-hearted and character-driven**, with:
    - Playful exchanges and personality in moment-to-moment play.
    - A more serious emotional thread underneath, focused on the partnership between the two protagonists.
